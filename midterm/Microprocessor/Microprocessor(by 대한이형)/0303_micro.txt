 01. 컴퓨터 시스템의 구성
-컴퓨터의 기본 구성
: 하드웨어, 소프트웨어

	1. 하드웨어
·중앙 처리 장치 = (마이크로)프로세서
-> ALU, 제어장치, 레지스터
·기억 장치 (주기억 장치, 보조 기억 장치)
·입출력 장치 (입력 장치, 출력 장치)
·시스템 버스: 중앙 처리 장치와 기억 장치 및 입출력 장치 사이에 정보를 교환하는 통로
-> 주소 버스, 데이터 버스, 제어 버스

 - 컴퓨터 시스템의 구성
: 주기억 장치는 중앙 처리 장치와 시스템 버스를 통해 연결된다.
 - 컴퓨터 시스템의 수행 기능
· 프로그램 실행
· 데이터 저장 ·데이터 이동 ·데이터 입출력
· 제어: 정해진 순서에 따라 실행

	 2. 소프트웨어
: 시스템 소프트웨어, 응용 소프트웨어

 -시스템 소프트웨어: 하드웨어를 관리하고 응용 소프트웨어를 실행하는 데 필요한 프로그램 -> 
· 운영체제
· 언어 번역 프로그램: 인터프리터, 컴파일러
· 장치 드라이버: 컴퓨터에 온라인으로 연결된 주변 장치를 제어하는 운영체제 모듈
· 링커: 하나의 프로그램으로 연결
· 로더: 저장 장치에 보관된 프로그램을 읽어 주기억 장치에 적재한 후 실행 가능 상태로

 -응용 소프트웨어 : 'Application'

	 3. 프로그램 처리 과정
고급언어 -> 어셈블리어 -> 기계어 순 변환

 *기계어 구조 예
1. LOAD A, X: 메모리 10번지 내용을 읽어 레지스터 A에 로드하라는 의미
2. ADD A, Y: 레지스터 A의 내용과 메모리 11번지의 내용을 더해 그 결과를 레지스터 A에 로드하라는 의미
3. STOR Z, A: 레지스터 A의 내용을 메모리 12번지에 저장하라는 의미
 * 연산 코드
·CPU가 수행할 연산을 지정하는 비트들
비트수 = 3이면, 연산 최대 수는 2^3 = 8개
 * 오퍼랜드
· 연산에 사용될 데이터가 저장되어 있는 기억 장치 주소
비트 수 = 5이면, 기억 장소의 최대 수 2^5
 * 프로그램과 데이터가 주기억 장치에 ㅇ저장되어 있는 형태 = 워드(word) 단위, 워드는 CPU에 의해 한 번에 처리될 수 있는 비트들의 그룹(8, 16, 32, 66? (64?)비트)

	4. 컴퓨터 시스템의 계층 레벨
·레벨 6: 사용자: 응용 프로그램들로 구성
·레벨 5: 고급언어: C/C++,JAVA,FORTRAN
고급언어->어셈블리어->기계어 순 변환
·레벨 4: 어셈블리어: 어셈블리어 명령어 1개가 기계어 명령어 1개로 변환
·레벨 3: 시스템 소프트웨어: 운영체제와 라이브러리 들로 구성
· 레벨 2: 기계어
· 레벨 1: 제어: 제어장치각 명령어들을 해독, 실행, 데이터들을 이동할 시간과 장소를 결정
